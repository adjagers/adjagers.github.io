Shopworks (schoolproject)

Shopworks is een mobile first applicatie waarin wij de shopware.com API hebben gebruikt om de data op te halen. De applicatie is gebouwd in React. Hier hebben we een swipe component aan toegevoegd. Hiermee swipe je de producten in je cart. Het is een e-commerce app met een specifieke usability.

Talen & frameworks

React, Redux en Scss

Website Annexbios (schoolproject)

Ter afsluiting van het 1e leerjaar heb ik met een projectteam een website gebouwd voor 1 van de locaties van de Annex bioscoop. Een ander projectteam zorgde voor de API. Wij hebben met die API data opgehaald met behulp van PHP voor de front-end en gebruik gemaakt van Bootstrap.

Functies website/app: Overzicht van de films, Filteren op categorie, Filteren op datum, Locatie weergave, Tickets bestellen, Film details.

Talen & frameworks

Html, Bootstrap en PHP

Pasveer app (commerciële opdracht)

Voor het adviseren van pensioenadviezen heeft Pasveer Pensioenadvies een kaartspel ontwikkeld. Dit kaartspel is bedoeld om op een speelse en leuke wijze de pensioenstatus van de cliënt weer te geven. Op basis hiervan verleent Pasveer passend advies aan de cliënt. De app die gemaakt is, is een digitalisering van dit kaartspel. Deze opdracht heb ik samen met een designer en ervaren ontwikkelaar uitgevoerd. Ik heb een deel van de applicatie zelfstandig gemaakt, onderandere de front-end functionaliteit.

Talen & frameworks

React, Laravel, Typescript en Scss

Easter-Wrestling app (schoolproject)

Easter-Wrestling is een mobiele applicatie voor worstel evenementen. Deelnemers en toeschouwers kunnen hier de aankomende wedstrijden zien. Je kunt zien hoe wedstrijden verlopen aan de hand van real-time status van de wedstrijden. Alle eindresultaten kunnen later teruggekeken worden via de app. Als projectteam hebben we de projectteam hebben we de front-end gebouwd. De API was al aanwezig.

Talen & frameworks

React, Laravel, Typescript en Scss

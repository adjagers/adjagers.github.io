Shopworks

Shopworks Frontend is a school project made using the shopware.com API.

Project info: Shopworks is een mobile first applicatie waarin wij de shopware.com API hebben gebruikt om de data op te halen. De applicatie is gebouwd in Reactjs hier we een swipe component toegevoegd waarbij als functie wanneer je een product swiped je de producten toevoegt aan je cart. Het was dus gebaseerd op een e-commerce app met een specifieke usability.

Annexbios

Shopworks Frontend is a school project made using the shopware.com API.

Project info: was het afsluit project van het 1e leer jaar tijdens het project werden 6 project teams opgesteld. Wij als project team moesten een website bouwen voor een van de panden van Annexbios. Een andere de project teams moest zorgen voor de data voor alle 6 de bioscoop panden. De rest van de teams maakten allemaal dezelfde website maar dan voor meerdere locaties. Aan de hand hiervan besloot het project team dat de data moest opleveren. Voor het maken van een API hierbij hebben alle project teams de data opgehaald wij als team met PHP en voor front-end bootstrap.

Talen/Technieken: Html, Bootstrap en PHP. Project groep: Luuk Lentjes Opdrachtgever: School project
